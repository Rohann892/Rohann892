### üìã Setting Up the Snake Animation

#### Step 1: Replace Your GitHub Username

In this `README.md`, replace all instances of:

- `Rohann892` ‚Üí Your actual GitHub username
- `your.email@gmail.com` ‚Üí Your email
- `your-username` ‚Üí Your GitHub handle
- `your-profile` ‚Üí Your LinkedIn profile URL
- `https://yourwebsite.com` ‚Üí Your portfolio website

#### Step 2: GitHub Actions Setup (Recommended)

1. **Already have the workflow file:** `.github/workflows/snake.yml` ‚úÖ
   - It's configured to run daily at midnight UTC
   - Also runs on every push

2. **Enable GitHub Actions:**
   - Go to Settings ‚Üí Actions ‚Üí General
   - Select "Allow all actions and reusable workflows"

3. **First Run:**
   - Go to Actions tab in your GitHub repo
   - Click "Generate Snake Animation" workflow
   - Click "Run workflow" to trigger it manually
   - Wait 2-3 minutes for completion

4. **Enable GitHub Pages (for snake animation):**
   - Go to Settings ‚Üí Pages
   - Source: Deploy from a branch
   - Branch: `output`
   - Folder: `/root`
   - Save

5. **Auto-Generation:**
   - Snake regenerates daily at midnight UTC
   - Also triggers on every push to repository

**GitHub Actions Workflow Configuration (.github/workflows/snake.yml):**

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 0 * * *" # Runs daily at midnight UTC
  workflow_dispatch: # Allows manual triggering

jobs:
  build:
    name: Generate snake animation
    runs-on: ubuntu-latest

    permissions:
      contents: write

    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: Platane/snk@v3
        id: snake-svg
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
